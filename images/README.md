# Images

The website uses real `<img>` slots that look for the files below. Until a file
exists, the page shows a styled colored panel with a caption instead of a broken
image — so the site looks intentional even before photos are added.

## Files the site looks for

| File | Where it appears | Suggested subject |
|---|---|---|
| `hero.jpg` | Hero (top right) | A warm, hopeful family or child, or a Puerto Rico coastal landscape |
| `families-1.jpg` | Mission gallery (left) | Families and children of Puerto Rico |
| `care-1.jpg` | Mission gallery (center) | A nurse practitioner / clinician caring for a patient |
| `island-1.jpg` | Mission gallery (right) | A Puerto Rico coastal community / landscape |

Recommended size: ~1600×1200 px (4:3) for the gallery, ~1600×1280 (5:4) for the hero.
Optimize/compress before committing (large images slow the page).

## Before you publish — please read

These are a public, donor-facing medical charity's pages. For any photo of a
person — especially a child:

1. **Use your own photographs, or properly licensed stock**, with a license that
   permits commercial/fundraising use.
2. **Get written consent / model releases**, and for children, consent from a
   parent or guardian.
3. **Do not present stock or unrelated images as actual patients or beneficiaries.**
   If an image is illustrative, caption it honestly.
4. Avoid flame/fire/ember imagery — per the brand guidance, lead with calm,
   healing, and hope, not the injury.
